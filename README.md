# BMI Calculator

A simple C++ program that calculates Body Mass Index (BMI) and categorizes weight status.

## How to Run
1. Compile: `g++ ConsoleApplication1.cpp -o bmi`
2. Run: `./bmi`

## Features
- Takes weight (kg) and height (cm) as input
- Calculates BMI using the formula: weight / (height in meters)Â²
- Categorizes result as underweight, normal, overweight, or obese
